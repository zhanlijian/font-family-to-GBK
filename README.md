css中文字体乱码解决方案：

把css编码和html页面编码统一起来。如果html页面是utf-8、css、js也统一成utf-8编码。如果html页面是gb2312、css、js也统一成gb2312编码。UTF-8和GB2312互转的最简单快捷的方法
